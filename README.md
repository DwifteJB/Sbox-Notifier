# Sbox-Notifier

Extremely simple sbox notifier and auto-joiner.

<img src="https://i.ibb.co/jDgMRrv/image.png" />

## How to use

### Tamper-monkey

Paste sbox-tampermonkey.js into a new tamper-monkey script and enable it

or

Press raw and it should auto-open.
